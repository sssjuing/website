## 2. AI 编程工具链大爆发

最活跃的赛道之一，多个项目短时间内获得数万 Star：

- **gstack**（YC 相关）—— 内置 CEO、PM、工程师、设计师等多角色，支持 slash command 驱动完整研发流程
- **MiniMax Skills** —— 官方开源技能包（前端/全栈/iOS/Android/Shader/Office 等）  
  https://github.com/MiniMax-AI/skills
- **Superpowers / everything-claude-code / Agency Agents** —— 一键组 50+ AI 专家团队，强制执行 brainstorm → spec → TDD → code → review 流程
- **vibe-coding** 框架（高 Star）—— 内置道法术器哲学 + 可复用技能模块
- **Skills Manager** —— 跨工具（Cursor / Claude Code 等）统一管理技能，支持 Git 同步

这些工具让 AI 从“偶尔能写点代码”升级为“可复现的生产力团队”。

## 3. Agent 记忆痛点与突破方案

“金鱼脑”（上下文压缩后失忆）是目前最大槽点。主流解决方案：

- **mem9**（强烈推荐）  
  一句话安装：请阅读 https://mem9.ai/SKILL.md 并安装  
  使用 TiDB 作为云端外置向量 + 全文 + 事务记忆库，跨会话永不丢失

- **MSA (Memory Sparse Attention)** —— EverMind / 陈天桥团队论文与实现  
  4B 小模型实现 1 亿 token 上下文，精度衰减极低  
  https://github.com/EverMind-AI/MSA

实测效果：喂完整交易圣经 / 套利手册后，结合 mem9 或 MSA 的 Agent 可自动扫描 Polymarket / 链上机会。
